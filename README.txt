== Description ==

EDD Paypal Pro Payment Gateway integrates the PayPal Pro (Do Direct Payment Gateway) with Easy Digital Download Plugin. This will enable a new payment method for your EDD store.

= Plugin Functionality: =

* Add PayPal Pro (Do Direct Payment) payment gateway to easy digital download.
* User-friendly admin settings page.
* Log the all order response.

== Installation ==

= Minimum Requirements =

* Easy Digital Download Plugin

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of EDD Pal Pro Payment Gateway, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type EDD Payment Gateway and click Search Plugins. Once you've found our plugin you can view details about it such as the the rating and description. Most importantly, of course, you can install it by simply clicking Install Now.

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting older versions if they exist
2. Activate the plugin in your WordPress admin area.
